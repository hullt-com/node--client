# Hullt.com | Client

[![Build Status](https://travis-ci.org/hullt-com/node--client.svg?branch=master)](https://travis-ci.org/hullt-com/node--client)

This Package allows you to:

- **authorize** Users
- **manage** Rights
- **read** and **write** User Settings
- **manage** Organizations
